const имя=значение


Описание:

Объявляет константу с заданным именем. В качестве значения может быть один из простых типов данных, а так же ссылка на другую константу.

Пример:

const c1=1 //объявлена константа с именем "c1" содержащая целочисленное значение 1
const c2=2.0 //объявлена константа с именем "c2" содержащая значение с плавающей точкой 2.0
const c3="three" //объявлена константа с именем "c3" содержащая строковое значение "three"
contt x=c1 //объявлена константа с именем "x" содержащая целочисленное значение 1 (оно было скопировано из константы c1)
