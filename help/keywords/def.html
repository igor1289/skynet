def имя_функции(параметр1, параметр2=выражение)


Описание:

Объявляет функцию с заданным именем. В качестве параметров так же можно задать выражение, которое будет вычисляться, если при вызове функции параметр опущен. Тело функции заканчивается ключевым словом end

Пример:

def f(x, y=1, z=2+2)
    return x+y+z
end
