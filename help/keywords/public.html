public имя=значение


Описание:

Объявляет публичную переменную с заданным именем. В качестве значения может быть один из простых типов данных, константы и любое выражение. Публичные переменные доступны из любого участка кода, а так же доступ к ним может осуществляться из исходной программы.

Пример:

public output="Hello world!"
