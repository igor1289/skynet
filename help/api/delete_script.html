SE_DeleteScript(Script.SE_Script)

Описание:

Удаляет скрипт из памяти. При удалении скрипта, его экземпляры (инстансы) не удаляются. Их необходимо удалить самостоятельно.

Параметры:

Script.SE_Script - Скрипт который необходимо удалить.

Пример:

SE_CompileScript("script.txt", "my_script.sky") Local Script.SE_Script=SE_LoadScriptBin.SE_Script("my_script.sky") SE_DeleteScript(Script)
