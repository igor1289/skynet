SE_FindPublic.SE_Public(Script.SE_Script, Name$)

Описание:

Осуществляет поиск нужной публичной переменной в скрипте.

Параметры:

Script.SE_Script - Скрипт в котором нужно найти переменную.
Name$ - Имя переменной.

Пример:

Local Script.SE_Script=SE_LoadScriptText("my_script.txt")
Local V.SE_Public=SE_FindPublic.SE_Public(Script, "my_var")
Print V\Value\ValueType
