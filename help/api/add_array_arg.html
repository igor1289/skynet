SE_AddArrayArg(Array.SE_Array)

Описание:

Позволяет установить передать аргумент перед вызовом пользовательской функции (Массив).

Параметры:

Array.SE_Array - Передаваемый массив.

Пример:

Local Script.SE_Script=SE_LoadScriptText("my_script.txt")
Local PrintAll.SE_FuncPtr=SE_FindFunc.SE_FuncPtr(Script, "print_all")
SE_AddNullArg()
SE_AddIntArg(1)
SE_AddFloatArg(3.5)
SE_AddStringArg("hello!")

SE_AddPointerArg(myValue)
SE_AddArrayArg(myArray)

SE_InvokeUserFunction(PrintAll)
