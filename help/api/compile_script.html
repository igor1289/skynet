SE_CompileScript(InFileName$, OutFileName$)

Описание:

Производит компиляцию скрипта в бинарный файл. Файлы такого типа загружаются быстрее. После компиляции, такой файл можно загрузить командой SE_LoadScriptBin()

Параметры:

InFileName$ - Иcходное имя файла.
OutFileName$ - Конечное имя файла.

Пример:

SE_CompileScript("script.txt", "my_script.sky") Local Script.SE_Script=SE_LoadScriptBin.SE_Script("my_script.sky")
