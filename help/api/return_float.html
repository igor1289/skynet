SE_ReturnFloat(Value#)

Описание:

Возвращает значение с плавающей точкой, после вызова глобальной функции.

Параметры:

Value# - Значение которое нужно вернуть в скрипт.

Пример:

Function SE_InvokeGlobalFunction(FunctionName$)
    If SE_BL_Math(FunctionName$) Then Return
    If SE_BL_Str(FunctionName$) Then Return
    If SE_BL_Array(FunctionName$) Then Return

    If FunctionName="sub"
        SE_ReturFloat(SE_FloatArg(0)/SE_FloatArg(1))
    EndIf
End Function
