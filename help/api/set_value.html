SE_SetValue(Pointer.SE_Value, Value$, ValueType)

Описание:

Позволяет установить значение указателя.

Параметры:

Pointer.SE_Value - Указатель, значение которого нужно установить.
Value$ - Значение (целое, с плавающей точкой, строка)
ValueType - Тип значения (SE_NULL, SE_INT, SE_FLOAT, SE_STRING)

Пример:

Function SE_InvokeGlobalFunction(FunctionName$)
    If SE_BL_Math(FunctionName$) Then Return
    If SE_BL_Str(FunctionName$) Then Return
    If SE_BL_Array(FunctionName$) Then Return

    If FunctionName="get_x"
        SE_SetValue(SE_PointerArg(0), "100", SE_INT)
    EndIf
End Function
