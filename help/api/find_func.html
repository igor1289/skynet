SE_FindFunc.SE_FuncPtr(Script.SE_Script, Name$)

Описание:

Осуществляет поиск нужной функции в скрипте.

Параметры:

Script.SE_Script - Скрипт в котором нужно найти функцию.
Name$ - Имя функции.

Пример:

Local Script.SE_Script=SE_LoadScriptText("my_script.txt")
Local F.SE_FuncPtr=SE_FindFunc.SE_FuncPtr(Script, "my_function")
SE_InvokeUserFunction(F)
