SE_CreateInstance.SE_Instance(Script.SE_Script)

Описание:

Создаёт экземпляр (инстанс) скрипта. Экземпляр позволяет обрабатывать, одним и тем же скриптом, разные наборы данных без повторной загрузки скрипта. Что существенно экономит память. При создании экземпляра, копируются переменные типа 'public', для последующей их обработки.

Параметры:

Script.SE_Script - Скрипт, экземпляр которого необходимо создать.
