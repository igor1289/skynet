SE_ArrayArg.SE_Value(Index)

Описание:

Позволяет получить аргумент вызванной глобальной функции по индексу. Возвращаемое значение является массивом переданным в качестве аргумента при вызове глобальной функции.

Параметры:

Index - Индекс аргумента. Должен быть в радиусе от [0, количество аргументов).

Пример:

Function SE_InvokeGlobalFunction(FunctionName$)
    If SE_BL_Math(FunctionName$) Then Return
    If SE_BL_Str(FunctionName$) Then Return
    If SE_BL_Array(FunctionName$) Then Return

    If FunctionName="set_vector"
        Vec.SE_Array=SE_ArrayArg(0)
        If Vec<>Null
            x#=SE_Array_GetValue(Vec, 0)
            y#=SE_Array_GetValue(Vec, 1)
            z#=SE_Array_GetValue(Vec, 2)
        EndIf     EndIf
End Function
