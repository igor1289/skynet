SE_InvokeGlobalFunction(FunctionName$)

Описание:

Позволяет выполнить пользовательскую функцию описанную в программе Blitz3D, вызванную при выполнении скрипта. Эту функцию нельзя вызывать, но по мере включения всё новых и новых функции, её необходимо модифицировать. Вызов функции осуществляется посредством проверки совпадения имени. Тоесть посредством IF и/или Select-Case. Как именно будет осуществляться проверка - пользователь решает сам. Единственное правило: чтобы обеспечить работу базовой библиотеки - вызовы функции лучше добавлять в конец функции (простите за тавтологию)

Примечание: Имена функций игнорируют регистр, поэтому если вы поставите сравнение, например 'If FunctionName="Print" Then Print("Hello world!")', то у вас никогда не выполнится функция Print().

Параметры:

FunctionName$ - Имя функции которую необходимо вызвать.
