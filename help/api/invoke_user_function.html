SE_InvokeUserFunction(FuncPtr.SE_FuncPtr, GCCollect=True)

Описание:

Позволяет выполнить пользовательскую функцию описанную в скрипте.

Параметры:

FuncPtr.SE_FuncPtr - указатель на функцию, которую необходимо вызвать. Его можно получить с помощью функции SE_FindFunc()
GCCollect=True - параметр очистки памяти. Необходим только виртуальной машине, чтобы не очистить память "раньше времени".
