SE_FloatArg#(Index, DefValue#=0)

Описание:

Позволяет получить аргумент вызванной глобальной функции по индексу. Особенность этой функции отличается в том, что она получает СТРОГО значения с плавающей точкой. Если индекс аргумента выходит за рамки количества аргументов или имеет другой тип значения, то возвращается значение по умолчанию указанное в качестве параметра DefValue#

Параметры:

Index - Индекс аргумента. Должен быть в радиусе от [0, количество аргументов).
DefValue#=0 - Значение возвращаемое по умолчанию.

Пример:

Function SE_InvokeGlobalFunction(FunctionName$)
    If SE_BL_Math(FunctionName$) Then Return
    If SE_BL_Str(FunctionName$) Then Return
    If SE_BL_Array(FunctionName$) Then Return

    If FunctionName="print"
        Print(SE_FloatArg(0, 1.5))
    EndIf
End Function
